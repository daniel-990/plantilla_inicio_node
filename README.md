
# Plantilla para el inicio de un proyecto en node

Esta plantilla es simple, las dependencias que tiene incorporadas son: 
 - express
 - mysql
 - ejs
 - config
 - body-parser


## Installation

Instalar el proyecto con npm

```bash
  npm install
  cd nombre-del-proyecto
```
